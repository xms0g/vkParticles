# vkParticles
A GPU-driven particle simulation written in modern C++ using the Vulkan API. The project demonstrates compute shaders for particle simulation and graphics pipelines for rendering thousands of particles efficiently while showcasing Vulkan synchronization, descriptor management, and modern rendering techniques.
